Myralix v10 · Simulation Loader

Inject sample candidate data into your browser's localStorage, then open the v10 Laporan to see it render with real scoring (and AI narratives if you're on claude.ai).

⚠️ Important: For localStorage to be shared between this loader and the Laporan, both files must be opened from the same origin (same domain + protocol). If you're opening them as local files, they need to be in the same folder. If hosted, both should be on the same subdomain (e.g., assess.myralix.id).

How to use

  1. Place this loader file and the 4 v10 Laporan HTML files in the same folder.
  2. Click "Load" on a candidate below — writes their data to myx-bat-{a,b,c,d}-v8.
  3. Click the "Open Laporan" link for that battery — opens the corresponding v10 Laporan with pre-loaded data.
  4. In the Laporan, go to the "Manager View" / "3-Pillar Bridge" section to see computed scores.
  5. If running inside claude.ai, AI narratives will also populate automatically.
