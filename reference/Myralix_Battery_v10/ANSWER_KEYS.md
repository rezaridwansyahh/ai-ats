## 4. SJT (Situational Judgment Test)
**22 skenario | 30 menit**

**6 Kompetensi:**

| Kode | Kompetensi | Jumlah | Max Score |
|------|------------|--------|-----------|
| KK | Kepemimpinan & Keputusan | 5 | 15 |
| KOM | Komunikasi & Kolaborasi | 4 | 12 |
| MK | Manajemen Konflik | 4 | 12 |
| OH | Orientasi Hasil | 3 | 9 |
| AD | Adaptabilitas & Problem Solving | 3 | 9 |
| IE | Integritas & Etika Kerja | 3 | 9 |

**Scoring System:**
Setiap skenario memiliki 4 opsi (A, B, C, D) dengan skor:
- **3 poin** = Respons terbaik
- **2 poin** = Respons baik
- **1 poin** = Respons kurang baik

**Answer Key (Best Response):**
```
Skenario 1-10:   A(KK), A(KK), A(KK), A(KK), A(KK), A(KOM), A(KOM), A(KOM), A(KOM), A(MK)
Skenario 11-20:  A(MK), A(MK), A(MK), A(OH), A(OH), A(OH), A(AD), A(AD), A(AD), A(IE)
Skenario 21-22:  A(IE), A(IE)
```

**Catatan:** Semua skenario memiliki opsi A sebagai respons terbaik (skor 3). Opsi B umumnya skor 2, C dan D skor 1-2 tergantung konteks.

**Profil SJT:**
- **66-60**: BJ (Bijaksana) - Unggul di semua dimensi
- **59-50**: EK (Eksekutor) atau KL (Kolaboratif)
- **49-40**: AE (Adaptif) - Fleksibel & solutif
- **<40**: BK (Berkembang) - Perlu pengembangan

# 📝 Catatan Teknis

## Normalisasi Jawaban (untuk matching)
```javascript
function normalizeAnswer(input) {
  return input.toString()
    .toLowerCase()
    .trim()
    .replace(/\s+/g, '')
    .replace(/[.,]/g, '');
}
```

**Contoh:**
- "YA", "ya", "Y", "y" → semua valid untuk soal yes/no
- "0.25", "1/4", "¼" → semua diterima untuk soal pecahan
- "2, 5" atau "2,5" → semua valid untuk soal multi-answer

## Special Cases
1. **GI Soal Input:** Case-insensitive, accept multiple formats
2. **EPPS/PAPI:** No correct answer, hanya pattern scoring
3. **SJT:** Weighted scoring (1-3 points)
4. **16PF:** Factor-based scoring dengan reversed items
5. **MSDT:** Frequency count per leadership style
